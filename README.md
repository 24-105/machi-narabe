# まちならべ

Project slug: `machi-narabe`

スマホ向けの街パズルです。7×7の盤面に建物を置き、同じ建物を3つ以上つなげて大きな建物を作ります。

## ゲーム内容

- 下の大きいカードに出ている建物を、空き地に置きます。
- 同じ建物が3つ以上つながると、次の建物に変わります。
- 変わった先でも3つそろうと、続けて合体します。
- 作った建物は、その後に出るカードにも混ざります。
- ゴミ屋敷はスコアが下がるじゃまな建物です。清掃車で消せます。
- 工事中はクレーンで直せます。建物を大きくする使い方もできます。
- 公園は屋台と組み合わせると、周りの建物が多いほどスコアが伸びます。
- 残り手数が0になるか、盤面が埋まると終了です。
- ミッションを達成すると、ボーナススコアと手数が増えます。
- 合体したり建物を置いたりしていると、お祭り・建設日和・清掃デー・商店街セールなどのイベントが起きます。
- おすすめマスは迷った時の目印です。慣れたら次のカードを見て、置き場所をずらします。
- ベストスコアはブラウザ内に保存します。

## SEO

- `index.html` に説明文、OG/Twitterカード、構造化データを設定しています。
- `robots.txt` は全ページのクロールを許可しています。
- リザルト画面に関連ゲーム「自転車ダッシュ」へのおすすめリンクを表示しています。
- 公開URLが決まったら、canonical URL と `sitemap.xml` を追加すると検索エンジンにより伝わりやすくなります。

## 起動

静的サイトなので、そのままGitHub Pagesで公開できます。

ローカル確認する場合:

```bash
python3 -m http.server 4387
```

ブラウザで `http://localhost:4387` を開きます。
